A Fragment implementation containing a PortalWebView for interacting with web-based blockchain applications.

Parameters

url

The initial URL to load in the WebView.

apiKey

The API key required for authentication with Portal services.

gatewayConfig

A mapping of chain IDs to their respective gateway URLs.

autoApprove

Determines if transactions should be automatically approved.

provider

The Provider responsible for blockchain interactions.

ethChainId

The Ethereum chain ID (default is 1 for mainnet).

eip6963Icon

The EIP-6963 compatible icon (default is a base64-encoded PNG).

eip6963Name

The name for EIP-6963 support (default is "Portal MPC Wallet").

eip6963Rdns

The reverse DNS identifier for EIP-6963 (default is "io.portalhq").

eip6963Uuid

The UUID identifier for EIP-6963 (default is "d73d7104-7e24-442b-913b-1147cd8e0325").
